موقع ويب كامل لمركز تجميل **لمسة جمال** في غزة، فلسطين 🇵🇸 — مبني بـ HTML وCSS وJavaScript خالص بدون أي مكتبات خارجية.

---

## 📋 نظرة عامة

موقع متعدد الصفحات يعرض خدمات مركز التجميل بتصميم عصري يدعم اللغة العربية (RTL) بشكل كامل، ويتضمن صفحات الرئيسية والخدمات والتواصل.

---

## ✨ المميزات

- 🌐 **دعم كامل للغة العربية** — اتجاه RTL مع خط Cairo من Google Fonts
- 📱 **تصميم متجاوب** — يعمل على جميع أحجام الشاشات
- ⚡ **شاشة تحميل** — Loader احترافي عند فتح الموقع
- 🎞️ **تأثيرات حركية** — انيميشن عند التمرير (Scroll Animations)
- 🔢 **عدادات متحركة** — إحصائيات المركز تُحسب بشكل ديناميكي
- 🎇 **جسيمات متحركة** — خلفية تفاعلية في قسم الهيرو
- 📬 **نموذج تواصل** — مع رسالة تأكيد عند الإرسال
- 🔝 **زر الرجوع للأعلى** — يظهر عند التمرير لأسفل
